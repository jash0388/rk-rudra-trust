# RK Rudra Chit Funds

A modern web application for RK Rudra Chit Funds — trusted chit fund services in Hyderabad.

## Tech Stack

- [Vite](https://vitejs.dev/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

The only requirement is having Node.js & npm installed — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Deployment

Build the production bundle and deploy to your hosting provider of choice:

```sh
npm run build
```

## Custom Domain

Configure your domain through your hosting provider's DNS settings and point it to the deployed app.
